install WAF
download mod security WAF from
https://www.apachehaus.com/downloads/mod_security2-2.9.6-2.4.x-x64-vs17.zip
unzip file and copy mob_security2.so to path C:\xampp\apache\modules                                                                  
copy file yajl.dll from path "C:\Users\xxxxx\Downloads\mod_security2-2.9.6-2.4.x-x64-vs17\Apache24\bin" to path C:\xampp\apache\bin                                           
create config file name modsecurity.conf in path C:\xampp\apache\conf 
and add this settings lines in modsecurity.conf
"SecAuditLog "C:/xampp/apache/logs/modsec_audit.log"
SecDebugLog "C:/xampp/apache/logs/modsec_debug.log"
SecRuleEngine DetectionOnly
SecRequestBodyAccess On
SecAuditEngine On
SecAuditLogParts ABC"
now create a new index in splunk name waf to send logs to splunk 
add this configuration lines in inputs.conf in path C:\Program Files\SplunkUniversalForwarder\etc\apps\SplunkUniversalForwarder\local
"[monitor://C:/xampp/apache/logs/modsec_audit.log]
disabled = false
index = waf
sourcetype = modsecurity:json"
add these lines "LoadModule unique_id_module modules/mod_unique_id.so
LoadModule security2_module modules/mod_security2.so
Include "C:/xampp/apache/conf/modsecurity.conf" in file httpd.conf in path C:\xampp\apache\conf
